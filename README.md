#Forge-and-Delivery

Jogo criado para a **Micro Jam 061**.

Um jogo onde você controla um anão ferreiro que viaja entre ilhas de barco para forjar armas e vender suas criações.

## 🎮 Controles

* **WASD / Setas** — Movimentação
* **X** — Interagir
* **Mouse** — Interações durante Time Events

## 🕹️ Gameplay

* Explore diferentes ilhas.
* Forje armas utilizando materiais disponíveis.
* Venda armas para conseguir recursos.
* Viaje de barco entre locais.

## 📌 Informações

**Jam:** Micro Jam 061
**Engine:** GameMaker Studio
**Gênero:** Crafting / Simulação / Aventura
